# Информационная система автоматизации продаж и сервисного обслуживания клиентов мотосалона АВАНТИС

Дипломная работа: **«Разработка информационной системы автоматизации продаж и сервисного обслуживания клиентов мотосалона АВАНТИС»**.

Система предназначена для автоматизации работы мотосалона: просмотра каталога мототехники, оформления заказов, обработки заявок на покупку, записи клиентов на сервисное обслуживание и управления обращениями через административную панель.

## Стек технологий

- Backend: PHP 8.2+, Laravel 12
- Frontend: Vue 3, TypeScript, Vue Router
- UI: Tailwind CSS 4, Vite
- База данных: SQLite, MySQL или PostgreSQL через Laravel migrations
- Авторизация: Laravel session auth для SPA

## Функционал клиента

- главная страница с акцентом на продажу, заявки и сервис AVANTIS;
- каталог мототехники с поиском, сортировкой и фильтрами по типу, бренду, году, цене, объему двигателя, мощности и наличию;
- карточка мотоцикла с крупным фото, характеристиками, преимуществами покупки и CTA-кнопками;
- корзина и оформление заказа;
- избранное и сравнение моделей;
- заявки на покупку, консультацию, уточнение наличия и предзаказ;
- страница сервисного обслуживания с услугами и формой записи;
- личный кабинет с личными данными, заказами, заявками на покупку, сервисными заявками и ссылкой на избранное;
- контактная форма обратной связи.

## Функционал администратора

- дашборд менеджера со статистикой и быстрым обзором новых обращений;
- управление товарами каталога: добавление, редактирование, удаление, статус наличия;
- группировка товаров по типу, бренду и наличию;
- просмотр заказов и изменение статусов;
- просмотр, фильтрация, удаление и изменение статусов заявок на покупку;
- просмотр, фильтрация, удаление и изменение статусов сервисных заявок;
- просмотр пользователей и их активности;
- просмотр сообщений обратной связи;
- цветные бейджи статусов, пустые состояния и адаптивные таблицы.

## Основные модули

- `Motorcycle` — мототехника и товары каталога;
- `Order` и `OrderItem` — оформленные заказы;
- `SalesRequest` — заявки на покупку, консультацию, уточнение наличия и предзаказ;
- `ServiceRequest` — заявки на сервисное обслуживание;
- `Favorite` — избранные товары пользователя;
- `ContactMessage` — сообщения с формы обратной связи;
- `User` — клиенты и администраторы.

## API-маршруты

- `POST /api/applications` — создание заявки на покупку;
- `GET /api/profile/applications` — заявки текущего пользователя;
- `GET /api/admin/applications` — заявки на покупку для администратора;
- `PATCH /api/admin/applications/{id}/status` — изменение статуса заявки;
- `DELETE /api/admin/applications/{id}` — удаление заявки;
- `POST /api/service-requests` — создание сервисной заявки;
- `GET /api/profile/service-requests` — сервисные заявки текущего пользователя;
- `GET /api/admin/service-requests` — сервисные заявки для администратора;
- `PATCH /api/admin/service-requests/{id}/status` — изменение статуса сервисной заявки;
- `DELETE /api/admin/service-requests/{id}` — удаление сервисной заявки.

## Запуск проекта

```bash
composer install
cp .env.example .env
php artisan key:generate
php artisan migrate --seed
npm install
npm run dev
php artisan serve
```

После запуска:

- клиентская часть: `http://127.0.0.1:8000`
- Vite dev server используется Laravel Vite plugin автоматически.

Сборка production-ассетов:

```bash
npm run build
```

Тесты Laravel:

```bash
php artisan test
```

## Статусы

Заявки на покупку:

- `new` — новая;
- `in_progress` — в работе;
- `approved` — согласована;
- `completed` — завершена;
- `cancelled` — отменена.

Сервисные заявки:

- `new` — новая;
- `confirmed` — подтверждена;
- `in_service` — техника в сервисе;
- `done` — готово;
- `cancelled` — отменена.

Заказы:

- `new` — новый;
- `processing` — в обработке;
- `completed` — завершен;
- `cancelled` — отменен.

## Визуальный стиль

Интерфейс оформлен в графитовой палитре с белым текстом и оранжево-красными акцентами AVANTIS. Дизайн ориентирован на мотосалон: техника, скорость, спорт, надежность, аккуратные карточки товара, статусные бейджи и деловые формы без лишней декоративности.
